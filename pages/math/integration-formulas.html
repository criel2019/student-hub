2. Integration by Substitution

When an integral contains a composite function, substitute u = g(x) so that du = gвҖІ(x)dx. The integral becomes one of the standard forms in terms of u.

Method: If the integrand has the pattern f(g(x)) В· gвҖІ(x), set u = g(x), du = gвҖІ(x)dx, then вҲ«f(g(x))gвҖІ(x)dx = вҲ«f(u)du вҖ” solve in u, then back-substitute.

Solved Examples

Find вҲ« sin(3x + 2) dx
Step 1: Let u = 3x + 2 вҶ’ du/dx = 3 вҶ’ dx = du/3 Step 2: вҲ« sin(u) В· (du/3) = (1/3) вҲ« sin u du Step 3: = (1/3)(вҲ’cos u) + C Step 4: Back-substitute: = вҲ’(1/3) cos(3x + 2) + C

Example 3
Find вҲ« вҲҡ(1 + cos 2x) dx
Step 1: Use identity: 1 + cos 2x = 2cosВІx Step 2: вҲҡ(1 + cos 2x) = вҲҡ(2cosВІx) = вҲҡ2 В· |cos x| Step 3: Assuming cos x вүҘ 0 (i.e., x вҲҲ [0, ПҖ/2]): = вҲҡ2 cos x Step 4: вҲ« вҲҡ2 cos x dx = вҲҡ2 sin x + C
Example 4 вҖ” Classic

3. Integration by Parts (IBP)

вҲ« u dv = uv вҲ’ вҲ« v du

Integration by parts is derived from the product rule of differentiation: d(uv)/dx = u(dv/dx) + v(du/dx). Integrating both sides gives the IBP formula.

LIATE Rule вҖ” Choose u in this priority order:
Logarithmic > Inverse Trigonometric > Algebraic (polynomials) > Trigonometric > Exponential

The function higher on the list becomes u; the other becomes dv. This choice usually simplifies the remaining integral.

Solved Examples

Example 1 вҖ” Algebraic Г— Exponential
Find вҲ« x В· eЛЈ dx
Choose: u = x (Algebraic), dv = eЛЈ dx (Exponential) Then: du = dx, v = eЛЈ IBP: вҲ« x eЛЈ dx = x В· eЛЈ вҲ’ вҲ« eЛЈ В· dx = x eЛЈ вҲ’ eЛЈ + C Answer: eЛЈ(x вҲ’ 1) + C
Example 2 вҖ” Algebraic Г— Trigonometric
Find вҲ« x В· cos x dx
Choose: u = x, dv = cos x dx Then: du = dx, v = sin x IBP: вҲ« x cos x dx = x sin x вҲ’ вҲ« sin x dx = x sin x вҲ’ (вҲ’cos x) + C Answer: x sin x + cos x + C
Example 3 вҖ” Degree 2 Algebraic Г— Exponential
Find вҲ« xВІ В· eЛЈ dx
Apply IBP twice. First pass: u = xВІ, dv = eЛЈdx вҶ’ du = 2x dx, v = eЛЈ вҲ« xВІ eЛЈ dx = xВІ eЛЈ вҲ’ вҲ« 2x eЛЈ dx = xВІ eЛЈ вҲ’ 2вҲ« x eЛЈ dx Second pass (from Example 1): вҲ« x eЛЈ dx = eЛЈ(xвҲ’1) Combine: xВІ eЛЈ вҲ’ 2В·eЛЈ(xвҲ’1) + C = eЛЈ(xВІ вҲ’ 2x + 2) + C
Example 4 вҖ” Logarithm alone
Find вҲ« ln x dx
Choose: u = ln x, dv = dx (i.e., dv = 1В·dx) Then: du = (1/x)dx, v = x IBP: вҲ« ln x dx = x ln x вҲ’ вҲ« x В· (1/x) dx = x ln x вҲ’ вҲ« 1 dx = x ln x вҲ’ x + C Answer: x(ln x вҲ’ 1) + C
Example 5 вҖ” Cyclical IBP (JEE Favourite!)
Find вҲ« eЛЈ В· sin x dx
Let I = вҲ« eЛЈ sin x dx First IBP: u = sin x, dv = eЛЈ dx вҶ’ du = cos x dx, v = eЛЈ I = eЛЈ sin x вҲ’ вҲ« eЛЈ cos x dx ... (i) Second IBP on вҲ« eЛЈ cos x dx: u = cos x, dv = eЛЈ dx вҶ’ du = вҲ’sin x dx, v = eЛЈ вҲ« eЛЈ cos x dx = eЛЈ cos x вҲ’ вҲ« eЛЈ(вҲ’sin x) dx = eЛЈ cos x + I ... (ii) Substitute (ii) into (i): I = eЛЈ sin x вҲ’ (eЛЈ cos x + I) I = eЛЈ sin x вҲ’ eЛЈ cos x вҲ’ I 2I = eЛЈ(sin x вҲ’ cos x) Answer: I = (eЛЈ/2)(sin x вҲ’ cos x) + C

4. Partial Fractions

Used when the integrand is a rational function P(x)/Q(x) where degree of P < degree of Q (if not, perform polynomial long division first).

Case 1 вҖ” Distinct Linear Factors

If Q(x) = (x вҲ’ aвӮҒ)(x вҲ’ aвӮӮ)В·В·В·(x вҲ’ aвӮҷ), then:

P(x)/Q(x) = AвӮҒ/(xвҲ’aвӮҒ) + AвӮӮ/(xвҲ’aвӮӮ) + В·В·В· + AвӮҷ/(xвҲ’aвӮҷ)

Case 2 вҖ” Repeated Linear Factor

For a factor (x вҲ’ a)вҒҝ, include all powers:

... = AвӮҒ/(xвҲ’a) + AвӮӮ/(xвҲ’a)ВІ + В·В·В· + AвӮҷ/(xвҲ’a)вҒҝ

Case 3 вҖ” Irreducible Quadratic Factor

For a factor (xВІ + bx + c) that doesn't factorise over reals:

... = (Ax + B)/(xВІ + bx + c)
Example 1 вҖ” Two distinct linear factors
Find вҲ« 1 / ((x+1)(x+2)) dx
Step 1 вҖ” Decompose: 1/((x+1)(x+2)) = A/(x+1) + B/(x+2) Step 2 вҖ” Multiply both sides by (x+1)(x+2): 1 = A(x+2) + B(x+1) Step 3 вҖ” Find A: Set x = вҲ’1: 1 = A(1) вҶ’ A = 1 Find B: Set x = вҲ’2: 1 = B(вҲ’1) вҶ’ B = вҲ’1 Step 4 вҖ” Integrate: вҲ« [1/(x+1) вҲ’ 1/(x+2)] dx = ln|x+1| вҲ’ ln|x+2| + C Answer: ln|(x+1)/(x+2)| + C
Example 2 вҖ” Linear Г— Quadratic
Find вҲ« (2x+3) / ((xвҲ’1)(xВІ+1)) dx
Decompose: (2x+3)/((xвҲ’1)(xВІ+1)) = A/(xвҲ’1) + (Bx+C)/(xВІ+1) Multiply out: 2x+3 = A(xВІ+1) + (Bx+C)(xвҲ’1) x = 1: 5 = A(2) вҶ’ A = 5/2 x = 0: 3 = A вҲ’ C вҶ’ C = A вҲ’ 3 = 5/2 вҲ’ 3 = вҲ’1/2 Compare xВІ coefficients: 0 = A + B вҶ’ B = вҲ’5/2 Integrate: вҲ« (5/2)/(xвҲ’1) dx + вҲ« (вҲ’5x/2 вҲ’ 1/2)/(xВІ+1) dx = (5/2)ln|xвҲ’1| вҲ’ (5/4)ln(xВІ+1) вҲ’ (1/2)tanвҒ»В№x + C
