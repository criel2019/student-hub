1. Trigonometric Ratios & Definitions

For a right-angled triangle with angle θ at vertex A (opposite side = a, adjacent side = b, hypotenuse = c):

sin θ = opposite / hypotenuse = a/c
cos θ = adjacent / hypotenuse = b/c
tan θ = opposite / adjacent = a/b = sin θ / cos θ
cot θ = adjacent / opposite = b/a = cos θ / sin θ
sec θ = hypotenuse / adjacent = c/b = 1 / cos θ
csc θ = hypotenuse / opposite = c/a = 1 / sin θ

Unit Circle Definition: For any angle θ, place a point P at (cos θ, sin θ) on the unit circle (radius 1). This extends trig ratios beyond 0° to 90° to all real angles.

4. Pythagorean Identities

Proof of sin²θ + cos²θ = 1:
By Pythagoras: opposite² + adjacent² = hypotenuse² → a² + b² = c²
Divide both sides by c²: (a/c)² + (b/c)² = 1 → sin²θ + cos²θ = 1 ✓

The Three Pythagorean Identities

Identity 1:
sin²θ + cos²θ = 1
⟹ sin²θ = 1 − cos²θ
⟹ cos²θ = 1 − sin²θ
Identity 2 (÷ cos²θ):
tan²θ + 1 = sec²θ
⟹ sec²θ − tan²θ = 1
⟹ (sec θ−tan θ)(sec θ+tan θ) = 1
Identity 3 (÷ sin²θ):   1 + cot²θ = csc²θ  ⟹ csc²θ − cot²θ = 1  ⟹ (csc θ−cot θ)(csc θ+cot θ) = 1
JEE tip: The factored forms (sec θ − tan θ)(sec θ + tan θ) = 1 and (csc θ − cot θ)(csc θ + cot θ) = 1 are used frequently to simplify rationalization problems.

5. Compound Angle Formulas

Sine and Cosine of Sum/Difference

sin(A+B) = sinA cosB + cosA sinB
sin(A−B) = sinA cosB − cosA sinB
cos(A+B) = cosA cosB − sinA sinB
cos(A−B) = cosA cosB + sinA sinB
Derivation of cos(A−B): Using unit circle geometry, place two unit vectors at angles A and B from the positive x-axis. The distance between their tips squared equals (cosA−cosB)²+(sinA−sinB)² by distance formula. By the law of cosines, this also equals 2−2cos(A−B). Equating: cos(A−B) = cosA cosB + sinA sinB ✓
All other compound formulas follow from this one.

Tangent of Sum/Difference

tan(A+B) = (tanA + tanB) / (1 − tanA tanB)
tan(A−B) = (tanA − tanB) / (1 + tanA tanB)
cot(A+B) = (cotA cotB − 1) / (cotB + cotA)
cot(A−B) = (cotA cotB + 1) / (cotB − cotA)
Example — Compound Angles
Prove that tan 75° = 2 + √3
tan 75° = tan(45° + 30°) = (tan 45° + tan 30°)/(1 − tan 45° · tan 30°) = (1 + 1/√3)/(1 − 1/√3) = (√3+1)/(√3−1) Rationalise: × (√3+1)/(√3+1) = (√3+1)²/(3−1) = (3+2√3+1)/2 = (4+2√3)/2 = 2+√3 ✓
Example — sin 105°
Find the exact value of sin 105°.
sin 105° = sin(60° + 45°) = sin60 cos45 + cos60 sin45 = (√3/2)(1/√2) + (1/2)(1/√2) = (√3 + 1)/(2√2) = (√6+√2)/4

8. t-Formulas (Weierstrass Substitution)

Let t = tan(A/2). Then:

sin A = 2t / (1+t²)
cos A = (1−t²) / (1+t²)
tan A = 2t / (1−t²)
1 − cos A = 2sin²(A/2) = 2t²/(1+t²)
Derivation: sin A = 2 sin(A/2) cos(A/2). Divide top and bottom by cos²(A/2): sin A = 2tan(A/2) / (1+tan²(A/2)) = 2t/(1+t²) ✓. Similarly cos A = (cos²(A/2)−sin²(A/2))/cos²(A/2)+sin²(A/2)) = (1−tan²(A/2))/(1+tan²(A/2)) = (1−t²)/(1+t²) ✓
Example — t-formula in equation
Solve: 2 sin x − cos x = 1 using t-formula.
Let t = tan(x/2). Then sin x = 2t/(1+t²), cos x = (1−t²)/(1+t²) 2·[2t/(1+t²)] − (1−t²)/(1+t²) = 1 Multiply through by (1+t²): 4t − (1−t²) = 1+t² 4t − 1 + t² = 1 + t² → 4t = 2 → t = 1/2 tan(x/2) = 1/2 → x = 2 tan⁻¹(1/2) + 2nπ Note: Also check t → ∞ case (x = π) separately, which gives 2(0)−(−1) = 1 ✓, so x = π + 2nπ is also a solution.

10. Triangle Formulas

For a triangle ABC with sides a (opposite A), b (opposite B), c (opposite C):

Sine Rule

a/sin A = b/sin B = c/sin C = 2R

where R is the circumradius (radius of circumscribed circle).

Cosine Rule

a² = b² + c² − 2bc cosA
b² = a² + c² − 2ac cosB
c² = a² + b² − 2ab cosC
cosA = (b²+c²−a²)/(2bc)

Projection Formulas

a = b cosC + c cosB
b = a cosC + c cosA

Area of Triangle

Δ = (1/2) ab sinC = (1/2) bc sinA = (1/2) ac sinB
Δ = √[s(s−a)(s−b)(s−c)]   [Heron's formula]
where s = (a+b+c)/2

Circumradius (R) and Inradius (r)

QuantityFormula
Circumradius RR = a/(2 sinA) = abc/(4Δ)
Inradius rr = Δ/s = (s−a)tan(A/2) = 4R sinA/2 sinB/2 sinC/2
tan(A/2)tan(A/2) = r/(s−a)
tan(B/2)tan(B/2) = r/(s−b)
tan(C/2)tan(C/2) = r/(s−c)
Napier's analogytan((A−B)/2) = (a−b)/(a+b) · cot(C/2)
Example — Triangle Calculation
In triangle ABC, a = 7, b = 8, c = 9. Find cos A and the area.
cosA = (b²+c²−a²)/(2bc) = (64+81−49)/(2·8·9) = 96/144 = 2/3 sinA = √(1 − 4/9) = √(5/9) = √5/3 Area = (1/2) bc sinA = (1/2)(8)(9)(√5/3) = 36√5/3 = 12√5 sq. units Using Heron's: s = (7+8+9)/2 = 12; Δ = √(12·5·4·3) = √720 = 12√5 ✓

11. 10 Solved JEE-Level Problems

Problem 1
Prove: sin 20° · sin 40° · sin 80° = √3/8
Use: sin θ · sin(60°−θ) · sin(60°+θ) = (1/4) sin 3θ Here θ = 20°: sin 20° · sin 40° · sin 80° = (1/4) sin 60° = (1/4)(√3/2) = √3/8 ✓
Problem 2
If tan α + tan β = a, cot α + cot β = b, find tan(α + β).
cot α + cot β = b → (1/tan α) + (1/tan β) = b → (tan α + tan β)/(tan α tan β) = b a / (tan α tan β) = b → tan α tan β = a/b tan(α+β) = (tan α + tan β)/(1 − tan α tan β) = a/(1 − a/b) = ab/(b−a)
Problem 3
Find the value of cos² 48° − sin² 12°.
Use: cos²A − sin²B = cos(A+B) cos(A−B) = cos(60°) cos(36°) = (1/2) cos 36° cos 36° = (1+√5)/4  (golden ratio connection) Answer = (1+√5)/8
Problem 4
If sin(A+B) = 1 and cos(A−B) = √3/2, find A and B (0 < A,B < π/2).
sin(A+B) = 1 → A+B = 90° ...(i) cos(A−B) = √3/2 → A−B = 30° ...(ii) Adding: 2A = 120° → A = 60°; then B = 30°
Problem 5
Evaluate: tan⁻¹(1/2) + tan⁻¹(1/5) + tan⁻¹(1/8)
First: tan⁻¹(1/2) + tan⁻¹(1/5) = tan⁻¹((1/2+1/5)/(1−1/10)) = tan⁻¹((7/10)/(9/10)) = tan⁻¹(7/9) Now: tan⁻¹(7/9) + tan⁻¹(1/8) = tan⁻¹((7/9+1/8)/(1−7/72)) = tan⁻¹((56+9)/(72)/(1−7/72)) = tan⁻¹((65/72)/(65/72)) = tan⁻¹(1) = π/4
Problem 6
If A + B + C = π, prove: sin 2A + sin 2B + sin 2C = 4 sinA sinB sinC.
LHS = sin 2A + sin 2B + sin 2C = 2 sin(A+B) cos(A−B) + sin 2C  [sum-to-product on first two] Since A+B+C = π, A+B = π−C, so sin(A+B) = sinC = 2 sinC cos(A−B) + 2 sinC cosC  [sin 2C = 2 sinC cosC] = 2 sinC [cos(A−B) + cosC] cosC = cos(π−A−B) = −cos(A+B) = 2 sinC [cos(A−B) − cos(A+B)] = 2 sinC · 2 sinA sinB = 4 sinA sinB sinC ✓
Problem 7
Solve: cos 2θ + cos θ = 0 for θ ∈ [0°, 360°]
cos 2θ + cos θ = 0 → (2cos²θ − 1) + cos θ = 0 2cos²θ + cosθ − 1 = 0 → (2cosθ − 1)(cosθ + 1) = 0 cosθ = 1/2 → θ = 60° or 300° cosθ = −1 → θ = 180° Solutions: θ = 60°, 180°, 300°
Problem 8
If tan θ = 3/4 and π < θ < 3π/2, find sin θ, cos θ.
θ is in Q3 (π to 3π/2), so sinθ < 0 and cosθ < 0. tan θ = 3/4 (positive in Q3 ✓). From a 3-4-5 triangle: opp=3, adj=4, hyp=5. sinθ = −3/5, cosθ = −4/5 (both negative in Q3)
Problem 9
In a triangle, show that: (a−b)² cos²(C/2) + (a+b)² sin²(C/2) = c²
LHS = (a−b)² cos²(C/2) + (a+b)² sin²(C/2) = (a²−2ab+b²)cos²(C/2) + (a²+2ab+b²)sin²(C/2) = (a²+b²)(cos²(C/2)+sin²(C/2)) + 2ab(sin²(C/2)−cos²(C/2)) = (a²+b²) − 2ab cosC = c²   [by cosine rule] ✓
Problem 10
Find all solutions of: sin x + sin 2x + sin 3x = 0
(sin x + sin 3x) + sin 2x = 0 2 sin 2x cos x + sin 2x = 0   [sum-to-product: sin x+sin 3x = 2sin 2x cos x] sin 2x (2 cos x + 1) = 0 Case 1: sin 2x = 0 → 2x = nπ → x = nπ/2 Case 2: cos x = −1/2 → x = 2π/3 + 2nπ or x = 4π/3 + 2nπ = ±2π/3 + 2nπ General solution: x = nπ/2 or x = 2nπ ± 2π/3, n ∈ ℤ
