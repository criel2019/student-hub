Example 1
A card is drawn from a standard deck of 52 cards. Find P(red or face card).
P(red) = 26/52 = 1/2; P(face) = 12/52 = 3/13; P(red AND face) = 6/52 = 3/26 P(red ∪ face) = 1/2 + 3/13 − 3/26 = 13/26 + 6/26 − 3/26 = 16/26 = 8/13
Example 2
Two dice are thrown. Find P(sum = 7) and P(at least one 6).
n(S) = 36. Sum = 7: (1,6),(2,5),(3,4),(4,3),(5,2),(6,1) → 6 outcomes. P = 6/36 = 1/6 P(at least one 6) = 1 − P(no 6) = 1 − (5/6)² = 1 − 25/36 = 11/36
Example 3
In a class, P(pass maths) = 2/3, P(pass physics) = 3/4, P(fail both) = 1/12. Find P(pass at least one).
P(fail both) = 1 − P(at least one) → P(at least one) = 1 − 1/12 = 11/12 Check: P(M∪P) = 2/3 + 3/4 − P(M∩P) = 11/12 → P(M∩P) = 8/12+9/12−11/12 = 6/12 = 1/2
Example 4
A bag has 3 red, 4 blue balls. Two balls are drawn without replacement. Find P(both red).
P(first red) = 3/7. Given first is red, P(second red) = 2/6 = 1/3. P(both red) = (3/7)(1/3) = 1/7 Using combinations: C(3,2)/C(7,2) = 3/21 = 1/7 ✓
Example 5
P(A) = 0.3, P(B) = 0.4, P(A∩B) = 0.12. Are A and B independent?
For independence: P(A∩B) should equal P(A)·P(B) = 0.3 × 0.4 = 0.12 P(A∩B) = 0.12 = 0.12 ✓. Yes, A and B are independent.

2. Conditional Probability & Multiplication Theorem

P(A|B) = P(A∩B) / P(B),    P(B) ≠ 0

Read as "probability of A given B has already occurred." The sample space reduces to B.

Multiplication Theorem

P(A∩B) = P(B) · P(A|B)
= P(A) · P(B|A)
Independent events:
P(A∩B) = P(A) · P(B)
⟹ P(A|B) = P(A)

Multiplication for 3 events

P(A∩B∩C) = P(A)·P(B|A)·P(C|A∩B)
Example 1
A bag has 5 white and 3 black balls. Two balls drawn without replacement. P(2nd white | 1st white)?
P(2nd white | 1st white) = P(both white)/P(1st white) = [C(5,2)/C(8,2)] / [5/8] = [10/28] / [5/8] = (10/28)·(8/5) = 80/140 = 4/7 Direct: After removing 1 white, bag has 4 white, 3 black → P = 4/7 ✓
Example 2
P(A) = 0.5, P(B|A) = 0.4, P(B|A') = 0.2. Find P(B).
P(B) = P(A)·P(B|A) + P(A')·P(B|A')   [Total Probability] = 0.5 × 0.4 + 0.5 × 0.2 = 0.2 + 0.1 = 0.3
Example 3
Find P(A∩B∩C) if P(A)=1/2, P(B|A)=1/3, P(C|A∩B)=1/4.
P(A∩B∩C) = P(A)·P(B|A)·P(C|A∩B) = (1/2)(1/3)(1/4) = 1/24
Example 4 — Independent events
A hits target with P=2/5, B with P=3/4. Both fire independently. P(target is hit)?
P(target hit) = 1 − P(both miss) = 1 − P(A misses)·P(B misses) = 1 − (3/5)(1/4) = 1 − 3/20 = 17/20
Example 5 — Law of Total Probability
Three machines M₁, M₂, M₃ produce 30%, 45%, 25% of a factory's output. Their defect rates are 2%, 3%, 2% respectively. A random item is picked; find P(defective).
P(D) = P(M₁)P(D|M₁) + P(M₂)P(D|M₂) + P(M₃)P(D|M₃) = 0.30×0.02 + 0.45×0.03 + 0.25×0.02 = 0.006 + 0.0135 + 0.005 = 0.0245

3. Bayes' Theorem

Statement & Derivation

Let A₁, A₂, ..., Aₙ be mutually exclusive and exhaustive events (partition of S), and let B be any event with P(B) > 0. Then:

P(Aᵢ|B) = P(Aᵢ) · P(B|Aᵢ) / [P(A₁)P(B|A₁) + P(A₂)P(B|A₂) + ··· + Pₙ)P(B|Aₙ)]

Derivation:

Numerator: P(Aᵢ ∩ B) = P(Aᵢ)·P(B|Aᵢ) [by multiplication theorem]
Denominator: P(B) = Σ P(Aⱼ)·P(B|Aⱼ) [Law of Total Probability]
Therefore: P(Aᵢ|B) = P(Aᵢ∩B)/P(B) = P(Aᵢ)P(B|Aᵢ) / Σ P(Aⱼ)P(B|Aⱼ) ✓

Prior probability: P(Aᵢ) — what we knew before observing B
Posterior probability: P(Aᵢ|B) — updated belief after observing B

Example 1 — Classic Bag Problem
Bag I has 3 red, 4 blue balls. Bag II has 5 red, 3 blue. A bag is chosen at random and one ball drawn. It is red. Find the probability it came from Bag I.
Define events: A₁ = Bag I chosen, A₂ = Bag II chosen, B = red ball drawn P(A₁) = P(A₂) = 1/2 P(B|A₁) = 3/7    P(B|A₂) = 5/8 P(B) = P(A₁)P(B|A₁) + P(A₂)P(B|A₂) = (1/2)(3/7) + (1/2)(5/8) = 3/14 + 5/16 = 24/112 + 35/112 = 59/112 Bayes: P(A₁|B) = P(A₁)P(B|A₁) / P(B) = (1/2·3/7) / (59/112) = (3/14) × (112/59) = (3 × 8)/59 = 24/59
Example 2 — Medical Testing (Sensitivity & Specificity)
A disease affects 1% of the population. A test has 95% sensitivity (true positive rate) and 90% specificity (true negative rate). If a person tests positive, what is P(has disease)?
Events: D = has disease, T = tests positive P(D) = 0.01, P(D') = 0.99 P(T|D) = 0.95 (sensitivity), P(T|D') = 1 − 0.90 = 0.10 (false positive rate) P(T) = P(D)P(T|D) + P(D')P(T|D') = 0.01 × 0.95 + 0.99 × 0.10 = 0.0095 + 0.099 = 0.1085 P(D|T) = P(D)P(T|D) / P(T) = 0.0095 / 0.1085 ≈ 0.0876 ≈ 8.76% Insight: Even with a positive test, there's only ~8.76% chance of having the disease due to low prevalence (base rate fallacy). This is why doctors re-test before diagnosing.
Example 3 — Factory Defect Problem
Three factories A, B, C produce 50%, 30%, 20% of a product. Defect rates: 1%, 2%, 3%. A defective item is found. Find P(it came from factory C).
P(A)=0.5, P(B)=0.3, P(C)=0.2 P(D|A)=0.01, P(D|B)=0.02, P(D|C)=0.03 P(D) = 0.5×0.01 + 0.3×0.02 + 0.2×0.03 = 0.005+0.006+0.006 = 0.017 P(C|D) = P(C)P(D|C)/P(D) = (0.2×0.03)/0.017 = 0.006/0.017 ≈ 0.353 (35.3%) Similarly: P(A|D) = 0.005/0.017 ≈ 29.4%; P(B|D) = 0.006/0.017 ≈ 35.3%
Example 4 — JEE Level
A letter is known to have come from TATANAGAR or CALCUTTA. On the postal stamp, only two consecutive letters "TA" are visible. Find the probability it came from TATANAGAR.
Event A₁ = from TATANAGAR, A₂ = from CALCUTTA; each equally likely: P(A₁)=P(A₂)=1/2 B = "TA" visible on stamp In TATANAGAR (9 letters), consecutive pairs: TA-AT-TA-AN-NA-AG-GA-AR = 8 pairs. "TA" appears at positions 1-2 and 3-4 → 2 times. P(B|A₁) = 2/8 = 1/4 In CALCUTTA (8 letters), pairs: CA-AL-LC-CU-UT-TT-TA = 7 pairs. "TA" appears once. P(B|A₂) = 1/7 P(B) = (1/2)(1/4) + (1/2)(1/7) = 1/8 + 1/14 = 7/56 + 4/56 = 11/56 P(A₁|B) = (1/2 × 1/4)/(11/56) = (1/8)/(11/56) = (1/8)(56/11) = 7/11 ≈ 7/11

4. Random Variables, E(X) and Var(X)

Types of Random Variables

Discrete: Countable outcomes (e.g., number of heads in 5 coin flips). Has a Probability Mass Function (PMF): P(X=x).
Continuous: Uncountably infinite outcomes (e.g., height, temperature). Has a Probability Density Function (PDF): f(x) where P(a≤X≤b) = ∫ₐᵇ f(x)dx.

Expected Value (Mean)

Variable TypeFormula
Discrete E(X)E(X) = Σ xᵢ · P(X=xᵢ) = Σ xᵢ pᵢ
Continuous E(X)E(X) = ∫₋∞^∞ x·f(x) dx
E(aX+b)= aE(X) + b   (linearity)
E(X²)= Σ xᵢ² · pᵢ

Variance & Standard Deviation

Var(X) = E(X²) − [E(X)]² = E[(X−μ)²]
σ = √Var(X)   (Standard Deviation)
Var(aX+b) = a²·Var(X)
Var(X+Y) = Var(X)+Var(Y) if X,Y independent
Example — Discrete Random Variable
X has PMF: P(X=0)=0.1, P(X=1)=0.3, P(X=2)=0.4, P(X=3)=0.2. Find E(X), E(X²), Var(X).
E(X) = 0(0.1)+1(0.3)+2(0.4)+3(0.2) = 0+0.3+0.8+0.6 = 1.7 E(X²) = 0²(0.1)+1²(0.3)+2²(0.4)+3²(0.2) = 0+0.3+1.6+1.8 = 3.7 Var(X) = E(X²)−[E(X)]² = 3.7−(1.7)² = 3.7−2.89 = 0.81 σ = √0.81 = 0.9

5. Binomial Distribution

When to Use Binomial Distribution?

  • Fixed number of trials: n
  • Each trial has exactly two outcomes: success (p) or failure (q = 1−p)
  • Trials are independent
  • p is constant across trials
P(X=r) = ⁿCᵣ · pʳ · qⁿ⁻ʳ,    r = 0, 1, 2, ..., n    (q = 1−p)
MeasureFormula
Meanμ = E(X) = np
VarianceVar(X) = npq = np(1−p)
Standard Deviationσ = √(npq)
ModeIf (n+1)p is integer: two modes; otherwise ⌊(n+1)p⌋
Important: Mean > Variance always in binomial (since q < 1). Mean = np, Variance = npq. The ratio Var/Mean = q = 1−p < 1.
Example 1
A fair coin is tossed 8 times. Find P(exactly 3 heads) and P(at least 6 heads).
n=8, p=1/2, q=1/2. P(X=r) = C(8,r)·(1/2)⁸ P(X=3) = C(8,3)·(1/2)⁸ = 56/256 = 7/32 P(X≥6) = P(6)+P(7)+P(8) = [C(8,6)+C(8,7)+C(8,8)]/256 = (28+8+1)/256 = 37/256
Example 2
A biased coin has P(heads) = 0.6. It is tossed 10 times. Find mean, variance, and P(exactly 4 heads).
n=10, p=0.6, q=0.4 Mean = np = 10 × 0.6 = 6 Variance = npq = 10 × 0.6 × 0.4 = 2.4; SD = √2.4 ≈ 1.55 P(X=4) = C(10,4)·(0.6)⁴·(0.4)⁶ = 210·0.1296·0.004096 ≈ 210·0.000531 ≈ 0.1115
Example 3
If the mean of a binomial distribution is 4 and variance is 3, find n and p.
np = 4 ...(i); npq = 3 ...(ii) Divide: q = 3/4 → p = 1/4 From (i): n·(1/4) = 4 → n = 16
Example 4
In a multiple choice test with 5 options each, a student guesses all 10 questions. Find the probability of getting more than 2 correct.
p = 1/5, q = 4/5, n = 10 P(X>2) = 1 − P(X≤2) = 1 − [P(0)+P(1)+P(2)] P(0) = (4/5)¹⁰ = 4¹⁰/5¹⁰ = 1048576/9765625 ≈ 0.1074 P(1) = 10·(1/5)·(4/5)⁹ ≈ 10·0.2·0.1342 ≈ 0.2684 P(2) = 45·(1/5)²·(4/5)⁸ ≈ 45·0.04·0.1678 ≈ 0.3020 P(X>2) = 1 − 0.1074 − 0.2684 − 0.3020 ≈ 0.3222 ≈ 32.2%
Example 5
X ~ B(n, 1/3). If P(X=1) = P(X=2), find n.
C(n,1)(1/3)¹(2/3)ⁿ⁻¹ = C(n,2)(1/3)²(2/3)ⁿ⁻² n·(1/3)·(2/3)ⁿ⁻¹ = [n(n−1)/2]·(1/9)·(2/3)ⁿ⁻² Divide: n·(1/3)·(2/3) = [n(n−1)/2]·(1/9) 2n/9 = n(n−1)/18 → 4 = n−1 → n = 5

6. Mean, Median & Mode

Ungrouped Data

MeasureFormula
Arithmetic Meanx̄ = (Σxᵢ)/n = (x₁+x₂+...+xₙ)/n
MedianSort data. If n is odd: middle value. If n is even: average of two middle values.
ModeValue that appears most frequently
Geometric MeanGM = (x₁·x₂···xₙ)^(1/n)
Harmonic MeanHM = n / (Σ 1/xᵢ)
RelationshipAM ≥ GM ≥ HM (for positive values)

Grouped Data Formulas

Mean (Direct method): x̄ = Σ(fᵢxᵢ) / Σfᵢ   where fᵢ = frequency, xᵢ = class midpoint
Mean (Assumed Mean / Shortcut method): x̄ = A + Σ(fᵢdᵢ) / Σfᵢ   where A = assumed mean, dᵢ = xᵢ − A
Mean (Step Deviation method): x̄ = A + h · Σ(fᵢuᵢ) / Σfᵢ   where uᵢ = (xᵢ−A)/h, h = class width
Median for grouped data (interpolation formula):
M = L + [(n/2 − cf) / f] × h
where L = lower boundary of median class, n = Σfᵢ, cf = cumulative frequency before median class, f = frequency of median class, h = class width
Mode for grouped data:
Mo = L + [(f₁−f₀) / (2f₁−f₀−f₂)] × h
where L = lower boundary of modal class, f₁ = highest frequency, f₀ = frequency of class preceding modal class, f₂ = frequency of class succeeding modal class, h = class width
Empirical Relationship (approximate, for moderately skewed data):
Mode ≈ 3·Median − 2·Mean
Mean − Mode ≈ 3(Mean − Median)
Example — Grouped Mean using Step Deviation
Find mean of: 0–10 (f=5), 10–20 (f=10), 20–30 (f=25), 30–40 (f=30), 40–50 (f=20), 50–60 (f=10).
Midpoints: 5, 15, 25, 35, 45, 55. Take A = 35, h = 10. uᵢ = (xᵢ−35)/10: −3, −2, −1, 0, 1, 2 fᵢuᵢ: 5(−3)+10(−2)+25(−1)+30(0)+20(1)+10(2) = −15−20−25+0+20+20 = −20 Σfᵢ = 100. Mean = 35 + 10·(−20)/100 = 35 − 2 = 33

7. Standard Deviation & Variance

MethodVariance Formula
Directσ² = Σ(xᵢ−x̄)²/n   (ungrouped)
Shortcutσ² = Σxᵢ²/n − (x̄)²
Grouped (direct)σ² = Σfᵢ(xᵢ−x̄)² / Σfᵢ
Grouped (step deviation)σ² = h² [Σfᵢuᵢ²/Σfᵢ − (Σfᵢuᵢ/Σfᵢ)²]
Sample variances² = Σ(xᵢ−x̄)²/(n−1)   (unbiased)

Combined Mean & Standard Deviation

Combined mean of two groups (n₁, x̄₁) and (n₂, x̄₂):
x̄₁₂ = (n₁x̄₁ + n₂x̄₂) / (n₁+n₂)
Combined variance:
σ² = [n₁(σ₁²+d₁²) + n₂(σ₂²+d₂²)] / (n₁+n₂)
where d₁ = x̄₁−x̄₁₂, d₂ = x̄₂−x̄₁₂

Coefficient of Variation (CV)

CV = (σ/x̄) × 100%

CV compares relative variability. A distribution with smaller CV is more consistent/uniform.

Example — Which batsman is more consistent?
Batsman A: mean = 50, SD = 10. Batsman B: mean = 40, SD = 6. Who is more consistent?
CV(A) = (10/50)×100 = 20% CV(B) = (6/40)×100 = 15% CV(B) < CV(A) → Batsman B is more consistent (less relative variation).

8. Correlation & Regression

Karl Pearson's Correlation Coefficient

r = Σ(xᵢ−x̄)(yᵢ−ȳ) / [√Σ(xᵢ−x̄)² · √Σ(yᵢ−ȳ)²] = Σdₓdᵧ / √(Σdₓ² · Σdᵧ²)

Where dₓ = xᵢ−x̄, dᵧ = yᵢ−ȳ. Range: −1 ≤ r ≤ 1.

  • r = +1: perfect positive linear correlation
  • r = −1: perfect negative linear correlation
  • r = 0: no linear correlation (could still be non-linear)
  • |r| > 0.7: strong correlation; 0.3–0.7: moderate; <0.3: weak

Spearman's Rank Correlation

rₛ = 1 − 6Σd²ᵢ / [n(n²−1)]

where dᵢ = difference between ranks of i-th pair. Used for ordinal data or non-normal distributions.

Lines of Regression

y on x (predict y from x):
y − ȳ = bᵧₓ(x − x̄)
bᵧₓ = r · σᵧ/σₓ = Σ(xᵢ−x̄)(yᵢ−ȳ)/Σ(xᵢ−x̄)²
x on y (predict x from y):
x − x̄ = bₓᵧ(y − ȳ)
bₓᵧ = r · σₓ/σᵧ
Key properties:
• bᵧₓ · bₓᵧ = r²   (product of regression coefficients)
• Both lines pass through (x̄, ȳ)
• r = ±√(bᵧₓ · bₓᵧ); sign same as regression coefficients
• If r=0: lines are perpendicular; if r=±1: both lines coincide
Example — Regression Lines
Given: x̄=6, ȳ=8, bᵧₓ=1.2, bₓᵧ=0.6. Find r and the two regression equations.
r² = bᵧₓ × bₓᵧ = 1.2 × 0.6 = 0.72 → r = √0.72 ≈ 0.849 (positive, since both b>0) y on x: y − 8 = 1.2(x − 6) → y = 1.2x − 7.2 + 8 → y = 1.2x + 0.8 x on y: x − 6 = 0.6(y − 8) → x = 0.6y − 4.8 + 6 → x = 0.6y + 1.2

9. 10 Solved JEE Problems

JEE Problem 1
A die is thrown. Event A = {1,2,3}, B = {2,3,4}. Find P(A|B) and P(B|A).
P(A∩B) = P({2,3}) = 2/6 = 1/3; P(A) = 3/6 = 1/2; P(B) = 3/6 = 1/2 P(A|B) = P(A∩B)/P(B) = (1/3)/(1/2) = 2/3 P(B|A) = P(A∩B)/P(A) = (1/3)/(1/2) = 2/3
JEE Problem 2
In a group of 400 people, 160 are smokers-non-drinkers, 100 are non-smokers-drinkers, and 240 are smokers. A person is chosen at random. Find P(drinker).
Smokers = 240. Of these, 240−160 = 80 are smoker+drinker. Non-smoker drinkers = 100. Total drinkers = 80+100 = 180. P(drinker) = 180/400 = 9/20
JEE Problem 3
Two cards are drawn from a pack of 52 without replacement. Find P(both aces).
P = C(4,2)/C(52,2) = 6/1326 = 1/221 Or: P = (4/52)(3/51) = 12/2652 = 1/221 ✓
JEE Problem 4
X ~ B(n, p). If E(X) = 6 and E(X²) = 38, find n and p.
E(X) = np = 6 ...(i); E(X²) = Var(X) + [E(X)]² = npq + 36 = 38 → npq = 2 From (i) and npq=2: q = 2/6 = 1/3 → p = 2/3 n × 2/3 = 6 → n = 9
JEE Problem 5
An unbiased die is thrown. Let X = number shown. Find E(X) and Var(X).
E(X) = (1+2+3+4+5+6)/6 = 21/6 = 7/2 = 3.5 E(X²) = (1+4+9+16+25+36)/6 = 91/6 Var(X) = 91/6 − (7/2)² = 91/6 − 49/4 = 182/12 − 147/12 = 35/12 ≈ 2.917
JEE Problem 6
A bag contains 4 red and 6 black balls. One ball is drawn, its colour noted and put back. Find P(after 3 draws: 2 red, 1 black).
With replacement → independent trials. p(red)=4/10=2/5, p(black)=3/5 X ~ B(3, 2/5). P(X=2) = C(3,2)(2/5)²(3/5) = 3·(4/25)·(3/5) = 36/125
JEE Problem 7
The probability that a bomb hits a target is 1/5. If 5 bombs are fired, find P(exactly 2 hit and target is destroyed by at least 2 hits).
P(exactly 2 hits) = C(5,2)(1/5)²(4/5)³ = 10·(1/25)·(64/125) = 640/3125 = 128/625
JEE Problem 8
If P(A) = 0.4, P(B) = 0.7, P(A∩B) = 0.3, find P(neither A nor B) and P(exactly one of A, B).
P(A∪B) = 0.4+0.7−0.3 = 0.8 P(neither) = 1−P(A∪B) = 1−0.8 = 0.2 P(exactly one) = P(A∪B)−P(A∩B) = 0.8−0.3 = 0.5 Or: P(A only) + P(B only) = (0.4−0.3)+(0.7−0.3) = 0.1+0.4 = 0.5 ✓
JEE Problem 9
The mean and SD of 20 observations are 10 and 2. Later it is found that observation 12 was misread as 8. Find the corrected mean and SD.
Wrong sum = 20×10 = 200. Corrected sum = 200−8+12 = 204. Corrected mean = 204/20 = 10.2 Wrong Σx² = n(σ²+x̄²) = 20(4+100) = 2080. Corrected Σx² = 2080−64+144 = 2160 Corrected σ² = 2160/20 − (10.2)² = 108 − 104.04 = 3.96; σ = √3.96 ≈ 1.99
JEE Problem 10
Two integers are chosen from 1 to 10 (inclusive) without replacement. Find P(both odd) and P(one odd, one even).
5 odd (1,3,5,7,9), 5 even (2,4,6,8,10). Total ways = C(10,2) = 45. P(both odd) = C(5,2)/45 = 10/45 = 2/9 P(one odd, one even) = C(5,1)·C(5,1)/45 = 25/45 = 5/9

10. Practice Problems (20 Questions)

Basic & Conditional Probability

Bayes' Theorem

  1. Urn A has 2 white 3 black; Urn B has 3 white 2 black. Urns are equally likely chosen. A white ball is drawn. Find P(from Urn B).
  2. In a certain city, 40% own a car and 30% own a bike. 10% own both. A person is a car owner. Find P(also owns bike).
  3. A test for a disease is 99% accurate (P(positive | disease) = 0.99, P(negative | no disease) = 0.99). If prevalence = 0.1%, find P(disease | positive test).
  4. Two factories A and B produce 60% and 40% of bulbs. Defect rate: A=2%, B=1%. A bulb is defective. P(from A)?

Binomial Distribution

  1. In 6 throws of a fair die, find P(getting 5 exactly twice).
  2. X ~ B(10, 1/4). Find P(X≥1) and E(X), Var(X).
  3. If X ~ B(n, 1/2) and P(X=3) = P(X=5), find n.
  4. Mean = 4, Variance = 3 for a binomial distribution. Find P(X=0).
