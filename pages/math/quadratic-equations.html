1. Standard Form & Terminology

ax² + bx + c = 0    (a ≠ 0)
TermMeaningExample: 2x² − 5x + 3 = 0
aLeading coefficient (coefficient of x²)a = 2
bMiddle coefficient (coefficient of x)b = −5
cConstant termc = 3
Roots / ZerosValues of x that satisfy the equation; denoted α, βα = 3/2, β = 1
Discriminant Db² − 4ac; determines nature of rootsD = 25 − 24 = 1 > 0
Why a ≠ 0 is required: If a = 0, the equation becomes bx + c = 0, which is a linear equation (at most 1 root, not 2). The term "quadratic" comes from the Latin "quadratus" (square).

2. Method 1 — Factoring

Express ax² + bx + c as a product of two linear factors (x − α)(x − β) = 0. Works best when roots are rational.

Key Step: Find two numbers p and q such that p + q = b and p · q = ac. Then split the middle term and factor by grouping.
Example 1 — Simple Monic
Solve: x² − 5x + 6 = 0
Find p, q: p + q = −5, p · q = 6 → p = −2, q = −3 x² − 2x − 3x + 6 = 0 → x(x−2) − 3(x−2) = 0 (x − 2)(x − 3) = 0 Answer: x = 2 or x = 3
Example 2 — Non-monic
Solve: 6x² + 11x − 10 = 0
a=6, b=11, c=−10. Need p+q=11, p·q=6·(−10)=−60 p=15, q=−4   (15+(−4)=11 ✓, 15·(−4)=−60 ✓) 6x² + 15x − 4x − 10 = 0 → 3x(2x+5) − 2(2x+5) = 0 (3x − 2)(2x + 5) = 0 Answer: x = 2/3 or x = −5/2
Example 3 — Difference of Squares
Solve: 4x² − 25 = 0
(2x)² − 5² = 0 → (2x − 5)(2x + 5) = 0 Answer: x = 5/2 or x = −5/2
Example 4 — Perfect Square
Solve: 9x² − 12x + 4 = 0
(3x)² − 2(3x)(2) + 2² = (3x − 2)² = 0 Answer: x = 2/3 (double root / equal roots)
Example 5 — Substitution then factoring
Solve: x⁴ − 13x² + 36 = 0
Let y = x²: y² − 13y + 36 = 0 (y − 4)(y − 9) = 0 → y = 4 or y = 9 x² = 4 → x = ±2; x² = 9 → x = ±3 Answer: x ∈ {−3, −2, 2, 3}

3. Method 2 — Completing the Square

The Technique: Write ax² + bx + c as a[(x + b/2a)² − (b/2a)² + c/a]. This isolates the squared term, making the equation solvable by taking square roots.
Example 1 — Standard
Solve: x² + 6x + 5 = 0 by completing the square
x² + 6x = −5 Add (6/2)² = 9 to both sides: x² + 6x + 9 = 4 (x + 3)² = 4 x + 3 = ±2 Answer: x = −1 or x = −5
Example 2 — Coefficient of x² ≠ 1
Solve: 2x² − 8x + 3 = 0
Step 1: Divide by 2: x² − 4x + 3/2 = 0 Step 2: x² − 4x = −3/2 Step 3: Add (4/2)² = 4: x² − 4x + 4 = −3/2 + 4 = 5/2 Step 4: (x − 2)² = 5/2 Step 5: x − 2 = ±√(5/2) = ±√10/2 Answer: x = 2 ± √10/2 = (4 ± √10)/2
Example 3 — No real roots
Solve: x² + 2x + 5 = 0
x² + 2x = −5 → x² + 2x + 1 = −4 → (x+1)² = −4 No real solution (square is non-negative, −4 is negative) Complex solution: x + 1 = ±2i → x = −1 ± 2i
Example 4 — Irrational coefficients
Solve: x² − 2√2 x + 1 = 0
x² − 2√2 x = −1 Add (√2)² = 2: (x − √2)² = 1 x − √2 = ±1 → x = √2 + 1 or x = √2 − 1
Example 5 — Derive the Quadratic Formula
Solve ax² + bx + c = 0 by completing the square.
Divide by a: x² + (b/a)x + c/a = 0 x² + (b/a)x = −c/a Add b²/(4a²): x² + (b/a)x + b²/(4a²) = b²/(4a²) − c/a (x + b/(2a))² = (b² − 4ac)/(4a²) x + b/(2a) = ±√(b² − 4ac)/(2a) x = (−b ± √(b² − 4ac)) / (2a) ← This IS the quadratic formula!

4. Method 3 — Quadratic Formula

x = (−b ± √(b² − 4ac)) / (2a)

This formula works for ALL quadratic equations. When in doubt, use it.

Example 1
Solve: 3x² + 7x − 6 = 0
a=3, b=7, c=−6. D = 49 − 4(3)(−6) = 49 + 72 = 121 x = (−7 ± √121)/(6) = (−7 ± 11)/6 x = (−7+11)/6 = 4/6 = 2/3   or   x = (−7−11)/6 = −18/6 = −3 Answer: x = 2/3 or x = −3
Example 2 — Equal roots
Solve: x² − 6x + 9 = 0
D = 36 − 36 = 0 x = (6 ± 0)/2 = 3 Answer: x = 3 (repeated root)
Example 3 — Complex roots
Solve: x² − 4x + 13 = 0
D = 16 − 52 = −36 x = (4 ± √(−36))/2 = (4 ± 6i)/2 Answer: x = 2 + 3i or x = 2 − 3i (complex conjugate pair)
Example 4 — Surd roots
Solve: x² − 4x + 1 = 0
D = 16 − 4 = 12 x = (4 ± √12)/2 = (4 ± 2√3)/2 = 2 ± √3 Answer: x = 2 + √3 or x = 2 − √3
Example 5 — Word problem
A ball is thrown upward with initial velocity 20 m/s from height 5 m. When does it hit the ground? (h = 5 + 20t − 5t²)
Set h = 0: 5 + 20t − 5t² = 0 → 5t² − 20t − 5 = 0 → t² − 4t − 1 = 0 t = (4 ± √(16+4))/2 = (4 ± √20)/2 = 2 ± √5 t = 2 + √5 ≈ 4.24 s   (negative root rejected) Answer: The ball hits the ground at t = 2 + √5 ≈ 4.24 seconds.

5. Method 4 — Graphical Method

The graph of y = ax² + bx + c is a parabola. The roots of ax² + bx + c = 0 are the x-intercepts (where the parabola crosses the x-axis).

  • If a > 0: parabola opens upward (cup shape, U)
  • If a < 0: parabola opens downward (cap shape, ∩)
  • Vertex: at x = −b/(2a), y = (4ac − b²)/(4a)
  • Axis of symmetry: x = −b/(2a)
  • Two x-intercepts → D > 0 (two distinct real roots)
  • One x-intercept (tangent) → D = 0 (equal roots)
  • No x-intercept → D < 0 (complex roots)

6. Discriminant D = b² − 4ac

D > 0 → Two distinct real roots
x₁ = (−b + √D)/(2a)   and   x₂ = (−b − √D)/(2a)
If D is a perfect square: roots are rational. Otherwise: irrational (surd) roots.
Example: x² − 5x + 6 = 0, D = 1 > 0 (perfect square) → x = 2, 3 (rational)
D = 0 → Two equal (repeated) real roots
x = −b/(2a)   (both roots are the same)
The parabola is tangent to the x-axis at the vertex.
Example: x² − 4x + 4 = 0, D = 0 → x = 2, 2
D < 0 → Two complex conjugate roots
x = (−b ± i√|D|)/(2a)   where i = √(−1)
The parabola does not intersect the x-axis.
Example: x² + x + 1 = 0, D = 1−4 = −3 → x = (−1 ± i√3)/2

7. Vieta's Formulas

If α and β are roots of ax² + bx + c = 0, then:

α + β = −b/a     αβ = c/a
Proof: If α, β are roots, then ax² + bx + c = a(x − α)(x − β) = a[x² − (α+β)x + αβ]. Comparing with ax² + bx + c: coefficient of x gives −a(α+β) = b → α+β = −b/a; constant term gives a·αβ = c → αβ = c/a. ✓

Forming an Equation from Given Roots

x² − (α+β)x + αβ = 0    or    x² − (sum)x + (product) = 0
Example — Form equation
Form a quadratic with roots 3 + √2 and 3 − √2.
Sum = (3+√2) + (3−√2) = 6 Product = (3+√2)(3−√2) = 9 − 2 = 7 Equation: x² − 6x + 7 = 0

Vertex Form

f(x) = a(x − h)² + k    where h = −b/(2a), k = (4ac−b²)/(4a)
Example — Maximum/Minimum
Find the minimum value of f(x) = 2x² − 8x + 11.
a = 2 > 0, so minimum exists. x = −b/(2a) = 8/4 = 2 Min value = f(2) = 2(4) − 8(2) + 11 = 8 − 16 + 11 = 3 Vertex form: 2(x − 2)² + 3 Minimum value = 3, achieved at x = 2.

12. Common Errors & JEE Traps

Error 1 — Forgetting to check D for "equal roots" condition:
For a quadratic to have equal roots, D = 0 (b² = 4ac). Many students confuse "equal roots" with "roots equal to zero".
Error 2 — Division by zero in Vieta's:
Vieta's formulas require a ≠ 0. Always verify the equation is truly quadratic before applying them.
Error 3 — Wrong sign in sum of roots:
For x² + bx + c = 0: α + β = −b, NOT b. The negative sign is often missed.
Error 4 — Extraneous roots from squaring:
When solving equations involving √(quadratic), squaring can introduce extraneous roots. Always verify in the original equation.
Error 5 — Assuming roots are always rational:
When D is not a perfect square, roots are irrational. Don't force rational approximations in exact-answer problems.
JEE Trap — Condition for common roots:
If a₁x²+b₁x+c₁ = 0 and a₂x²+b₂x+c₂ = 0 have a common root α, then:
(a₁c₂ − a₂c₁)² = (b₁c₂ − b₂c₁)(a₁b₂ − a₂b₁) — this condition is frequently tested.

13. 10 Solved JEE-Level Problems

JEE Problem 1
If one root of x² − px + q = 0 is twice the other, find the relationship between p and q.
Let roots be α and 2α. Sum: α + 2α = p → 3α = p → α = p/3 Product: α · 2α = q → 2α² = q → 2(p/3)² = q → 2p²/9 = q Relationship: 2p² = 9q
JEE Problem 2
Find all values of k for which x² − kx + k + 3 = 0 has real roots with one root greater than 3 and other less than 1.
Let f(x) = x² − kx + k + 3. Parabola opens up (a=1>0). For one root > 3, other < 1: we need f(1) < 0 AND f(3) < 0. f(1) = 1 − k + k + 3 = 4 > 0. This is always positive! So f(1) < 0 has no solution. The condition cannot be satisfied. Answer: No such k exists. (We need f(1) and f(3) to have the same sign since both 1 and 3 are on the same "outside" region.)
JEE Problem 3
If α, β are roots of x² − 2x + 3 = 0, form the equation with roots α + 2, β + 2.
α + β = 2, αβ = 3. New sum = (α+2) + (β+2) = α + β + 4 = 6 New product = (α+2)(β+2) = αβ + 2(α+β) + 4 = 3 + 4 + 4 = 11 New equation: x² − 6x + 11 = 0
JEE Problem 4
Find the range of values of a for which f(x) = x² − ax + 1 − 2a² > 0 for all real x.
For f(x) > 0 for all x, need a > 0 (already, since coefficient of x² is 1) AND D < 0. D = a² − 4(1−2a²) = a² − 4 + 8a² = 9a² − 4 < 0 9a² < 4 → a² < 4/9 → |a| < 2/3 → −2/3 < a < 2/3 Answer: a ∈ (−2/3, 2/3)
JEE Problem 5
Let α, β be roots of x² − x − 1 = 0. Find αⁿ + βⁿ.
α + β = 1, αβ = −1. Note: α² = α + 1 and β² = β + 1 (from equation). Let Sₙ = αⁿ + βⁿ. Then Sₙ = (α+β)Sₙ₋₁ − αβ·Sₙ₋₂ = Sₙ₋₁ + Sₙ₋₂ S₀ = 2, S₁ = 1. This is the Lucas sequence: 2, 1, 3, 4, 7, 11, ... The recurrence Sₙ = Sₙ₋₁ + Sₙ₋₂ is the Fibonacci-type relation.
JEE Problem 6
If p and q are roots of x² + px + q = 0, find p and q.
By Vieta's: p + q = −p and pq = q. From pq = q: q(p−1) = 0 → q = 0 or p = 1. Case 1: q = 0. Then p + 0 = −p → 2p = 0 → p = 0. Equation: x² = 0. Roots 0, 0 — but then the roots are 0 and 0, not p=0 and q=0... consistent! Case 2: p = 1. Then 1 + q = −1 → q = −2. Check: pq = (1)(−2) = −2 = q ✓. Equation: x²+x−2=0, roots 1, −2 ✓ Answer: (p, q) = (0, 0) or (p, q) = (1, −2)
JEE Problem 7
Find k if the equation (k−2)x² + 8x + k+4 = 0 has real and equal roots.
For equal roots: D = 0 → 64 − 4(k−2)(k+4) = 0 64 = 4(k−2)(k+4) = 4(k²+2k−8) 16 = k² + 2k − 8 → k² + 2k − 24 = 0 → (k+6)(k−4) = 0 k = −6 or k = 4. But if k = 2, not quadratic — check: k=−6 ✓, k=4 ✓. Answer: k = −6 or k = 4
JEE Problem 8
For what value of m does mx² + (m+1)x + 2 = 0 have equal roots?
D = (m+1)² − 8m = 0 → m² + 2m + 1 − 8m = 0 → m² − 6m + 1 = 0 m = (6 ± √32)/2 = 3 ± 2√2 Answer: m = 3 + 2√2 or m = 3 − 2√2
JEE Problem 9
If ax² + bx + c = 0 and bx² + cx + a = 0 have a common root, prove a³ + b³ + c³ = 3abc.
Let α be the common root. Then: aα² + bα + c = 0 ...(i) and bα² + cα + a = 0 ...(ii) (i)×b − (ii)×a: b²α + bc − acα − a² = 0 → α(b²−ac) = a²−bc → α = (a²−bc)/(b²−ac) ...(iii) (i)×c − (ii)×b: acα² + bcα + c² − b²α² − bcα − ab = 0 → α²(ac−b²) = ab−c² → α² = (ab−c²)/(ac−b²) ...(iv) From (iii) and (iv): α² = α² → [(a²−bc)/(b²−ac)]² = (ab−c²)/(ac−b²) After simplification (cross multiply and expand): a³+b³+c³−3abc = 0 ✓
JEE Problem 10
Find the quadratic equation whose roots are the squares of the roots of x² + x − 2 = 0.
Roots of original: α, β where α+β = −1, αβ = −2 New roots: α², β². New sum = α²+β² = (α+β)²−2αβ = 1+4 = 5 New product = (αβ)² = (−2)² = 4 New equation: x² − 5x + 4 = 0 Check: roots are 1 and 4 = (−1)² and (−2)² ✓ (original roots: 1 and −2) ✓
