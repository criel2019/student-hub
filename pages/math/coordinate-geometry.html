1. Distance, Section, Midpoint & Area Formulas

FormulaExpressionNotes
Distance between P₁(x₁,y₁) and P₂(x₂,y₂) d = √[(x₂−x₁)² + (y₂−y₁)²] Pythagoras in the plane
Midpoint of P₁P₂ M = ((x₁+x₂)/2, (y₁+y₂)/2)
Section formula (internal division) — ratio m:n P = ((mx₂+nx₁)/(m+n), (my₂+ny₁)/(m+n)) Midpoint is m=n=1
Section formula (external division) — ratio m:n P = ((mx₂−nx₁)/(m−n), (my₂−ny₁)/(m−n)) m ≠ n
Area of triangle with vertices (x₁,y₁), (x₂,y₂), (x₃,y₃) A = (1/2)|x₁(y₂−y₃) + x₂(y₃−y₁) + x₃(y₁−y₂)| Absolute value ensures positivity
Collinearity condition x₁(y₂−y₃) + x₂(y₃−y₁) + x₃(y₁−y₂) = 0 Area = 0
Example 1
A(1,2), B(4,6). Find midpoint M, distance AB, and point P that divides AB in ratio 2:1.
Midpoint: M = ((1+4)/2, (2+6)/2) = (2.5, 4) Distance: AB = √[(4−1)²+(6−2)²] = √[9+16] = √25 = 5 P (2:1): = ((2·4+1·1)/(3), (2·6+1·2)/3) = (9/3, 14/3) = (3, 14/3)

Key Results for Straight Lines

PropertyFormula
Slope of line ax+by+c=0m = −a/b
Angle between lines with slopes m₁, m₂tan θ = |(m₁−m₂)/(1+m₁m₂)|
Parallel lines conditionm₁ = m₂ (slopes equal)
Perpendicular lines conditionm₁ · m₂ = −1
Distance from point (x₀,y₀) to line ax+by+c=0d = |ax₀+by₀+c| / √(a²+b²)
Distance between parallel lines ax+by+c₁=0 and ax+by+c₂=0d = |c₁−c₂| / √(a²+b²)
Foot of perpendicular from (x₁,y₁) to ax+by+c=0(x₁−a·λ, y₁−b·λ) where λ = (ax₁+by₁+c)/(a²+b²)
Image (reflection) of (x₁,y₁) in ax+by+c=0(x₁−2aλ, y₁−2bλ) where λ = (ax₁+by₁+c)/(a²+b²)

Family of Lines

Any line through the intersection of L₁: a₁x+b₁y+c₁=0 and L₂: a₂x+b₂y+c₂=0 is:
L₁ + λL₂ = 0, i.e., (a₁x+b₁y+c₁) + λ(a₂x+b₂y+c₂) = 0 for some λ.
Choose λ to satisfy any additional condition on the line.

Pair of Straight Lines through Origin

The equation ax² + 2hxy + by² = 0 represents a pair of lines through the origin. If slopes m₁, m₂:

m₁ + m₂ = −2h/b
m₁ · m₂ = a/b
Angle between them:
tan θ = 2√(h²−ab)/(a+b)

The pair of angle bisectors of ax²+2hxy+by²=0 is: (x²−y²)/(a−b) = xy/h

Example 1 — Distance and Perpendicular
Find the distance from (3, −4) to the line 3x − 4y + 10 = 0 and the foot of the perpendicular.
Distance = |3(3) − 4(−4) + 10| / √(9+16) = |9+16+10|/5 = 35/5 = 7 units λ = (ax₁+by₁+c)/(a²+b²) = (9+16+10)/25 = 35/25 = 7/5 Foot = (3 − 3·(7/5), −4 − (−4)·(7/5)) = (3−21/5, −4+28/5) = (−6/5, 8/5)
Example 2 — Family of Lines
Find the line through the intersection of 2x+y−3=0 and x−3y+4=0 that passes through (1,1).
Family: (2x+y−3) + λ(x−3y+4) = 0 Substitute (1,1): (2+1−3) + λ(1−3+4) = 0 → 0 + 2λ = 0 → λ = 0 Answer: 2x + y − 3 = 0 (the first line itself passes through (1,1)) Verify: 2(1)+1−3 = 0 ✓
Example 3 — Image in a line
Find the image of point (1, 2) in the line x − y + 1 = 0.
a=1, b=−1, c=1, (x₁,y₁)=(1,2) λ = (1·1 + (−1)·2 + 1)/(1+1) = (1−2+1)/2 = 0 Image = (1−2(1)(0), 2−2(−1)(0)) = (1, 2) — the point is ON the line! Check: 1−2+1 = 0 ✓. So the image of a point on the line is itself.
Example 4 — Angle bisectors
Find the equations of the bisectors of angles between 3x−4y+7=0 and 12x+5y−2=0.
Angle bisectors: (3x−4y+7)/√(9+16) = ±(12x+5y−2)/√(144+25) (3x−4y+7)/5 = ±(12x+5y−2)/13 +ve: 13(3x−4y+7) = 5(12x+5y−2) → 39x−52y+91 = 60x+25y−10 → 21x+77y = 101 → (simplify) −ve: 13(3x−4y+7) = −5(12x+5y−2) → 39x−52y+91 = −60x−25y+10 → 99x−27y+81=0 → 11x−3y+9=0
Examples 5–10 (condensed)
Area of triangle formed by lines x+y=1, 2x−y=4, x−2y+1=0.
Find the three intersection points: L1∩L2: x+y=1 and 2x−y=4 → 3x=5 → x=5/3, y=−2/3 → P₁=(5/3,−2/3) L2∩L3: 2x−y=4 and x−2y=−1 → from 1st: y=2x−4; sub: x−2(2x−4)=−1 → x=3, y=2 → P₂=(3,2) L1∩L3: x+y=1 and x−2y=−1 → 3y=2 → y=2/3, x=1/3 → P₃=(1/3,2/3) Area = (1/2)|5/3(2−2/3)+3(2/3+2/3)+1/3(−2/3−2)| = (1/2)|5/3·4/3 + 3·4/3 + 1/3·(−8/3)| = (1/2)|20/9 + 4 − 8/9| = (1/2)|12/9+4| = (1/2)|16/3| = 8/3

4. Circles

Standard Equations of a Circle
Centre (0,0), radius rx² + y² = r²
Centre (h,k), radius r(x−h)² + (y−k)² = r²
General equationx² + y² + 2gx + 2fy + c = 0
Centre of general circle(−g, −f)
Radius of general circler = √(g²+f²−c)   [real if g²+f²−c > 0]
Diametric form (endpoints A(x₁,y₁), B(x₂,y₂))(x−x₁)(x−x₂)+(y−y₁)(y−y₂)=0

Example 1 — Circle through three points
Find the circle passing through (1,0), (0,1) and (0,0).
General: x²+y²+2gx+2fy+c=0. Through (0,0): c=0. Through (1,0): 1+2g+c=0 → 1+2g=0 → g=−1/2 Through (0,1): 1+2f+c=0 → f=−1/2 Circle: x²+y²−x−y=0. Centre=(1/2,1/2), r=√(1/4+1/4)=1/√2
Example 2 — Length of tangent
Find the length of the tangent from (5,4) to the circle x²+y²−3x+4y+1=0.
L = √(x₁²+y₁²+2gx₁+2fy₁+c) with 2g=−3, 2f=4, c=1, (x₁,y₁)=(5,4) = √(25+16−15+16+1) = √43
Example 3 — Radical Axis
Find the radical axis of x²+y²−3x+4y−1=0 and x²+y²−x+2y+3=0.
S₁−S₂ = (−3x+4y−1)−(−x+2y+3) = −2x+2y−4 = 0 → x−y+2=0 Radical axis: x − y + 2 = 0
Example 4 — Family of circles
Write the family of circles through the intersection of x²+y²−4=0 and x+y−2=0.
Family: (x²+y²−4) + λ(x+y−2) = 0 = x²+y²+λx+λy−(4+2λ)=0 Centre=(−λ/2, −λ/2), r²=λ²/4+λ²/4+(4+2λ)=λ²/2+4+2λ

Parametric Form (y² = 4ax)

Any point on y² = 4ax: (at², 2at) for parameter t ∈ ℝ.

Important Lines for Parabola y² = 4ax

LineEquation
Tangent at (at², 2at)ty = x + at²
Tangent at point (x₁,y₁)yy₁ = 2a(x + x₁)
Normal at (at², 2at)y = −tx + 2at + at³
Chord joining t₁ and t₂y(t₁+t₂) = 2x + 2at₁t₂
Focal chord (t₁t₂ = −1)t₁t₂ = −1 always for focal chord
Chord of contact from (x₁,y₁)yy₁ = 2a(x+x₁)
Focal Chord Property: If one end of a focal chord is at parameter t, the other end is at parameter −1/t.
Sum of reciprocals of distances: If SP and SQ are the two focal segments (S = focus), then 1/SP + 1/SQ = 1/a.
Example 1 — Tangent and Normal
Find the tangent and normal to y² = 8x at the point (2, 4).
Here 4a = 8, so a = 2. Check: (2,4) → 4² = 16, 8·2 = 16 ✓. So (x₁,y₁)=(2,4). Tangent: yy₁ = 2a(x+x₁) → 4y = 4(x+2) → y = x + 2 Slope of tangent = 1. Slope of normal = −1. Normal: y−4 = −1(x−2) → y = −x + 6 → x + y = 6
Example 2 — Focal chord
For y² = 4x, if one end of a focal chord is at (1/4, −1), find the other end.
a=1. Point (1/4, −1) = (at², 2at) → 2at = −1 → t = −1/2. Other end: t₂ = −1/t₁ = 2. Other end point: (at₂², 2at₂) = (4, 4).
Example 3 — Parabola and straight line
Find the condition for y = mx + c to be tangent to y² = 4ax.
Substitute y = mx + c into y² = 4ax: (mx+c)² = 4ax → m²x²+(2mc−4a)x+c²=0 For tangency: discriminant = 0 → (2mc−4a)² − 4m²c² = 0 4m²c²−16amc+16a²−4m²c² = 0 → 16a(a−mc) = 0 → c = a/m Condition: c = a/m Tangent: y = mx + a/m (standard tangent to y²=4ax with slope m)

6. Ellipse

Standard Ellipse: x²/a² + y²/b² = 1 (a > b > 0)
Major axisAlong x-axis, length 2a
Minor axisAlong y-axis, length 2b
Foci(±c, 0) where c² = a² − b²
Vertices(±a, 0) and (0, ±b)
Eccentricitye = c/a   (0 < e < 1)
Directricesx = ±a/e = ±a²/c
Latus rectumx = ±c, length = 2b²/a
Focal radiiSP = a − ex, S'P = a + ex (S,S' are foci)
SP + S'P = 2a(definition of ellipse)

Parametric Form

Any point on the ellipse: (a cosθ, b sinθ) for θ ∈ [0, 2π).

Lines and Tangents

LineEquation
Tangent at (x₁,y₁)xx₁/a² + yy₁/b² = 1
Tangent in slope form (slope m)y = mx ± √(a²m² + b²)
Normal at (x₁,y₁)a²x/x₁ − b²y/y₁ = a²−b²
Chord of contact from (x₁,y₁)xx₁/a² + yy₁/b² = 1
Example 1 — Foci and eccentricity
For x²/25 + y²/16 = 1, find foci, eccentricity, and length of latus rectum.
a² = 25, b² = 16, so a=5, b=4. c² = 25−16 = 9 → c = 3. Foci: (±3, 0). Eccentricity: e = 3/5. Latus rectum length = 2b²/a = 2(16)/5 = 32/5.
Example 2 — Tangent to ellipse
Find the tangent to x²/9 + y²/4 = 1 at the point (√3, 2/√3)... or find tangent with slope 1.
Tangent with slope m=1: y = x ± √(a²m²+b²) = x ± √(9+4) = x ± √13 Tangents: y = x + √13 and y = x − √13
Example 3 — Area of ellipse
Show that the area enclosed by x²/a² + y²/b² = 1 is πab.
y = (b/a)√(a²−x²). Total area = 4·∫₀ᵃ (b/a)√(a²−x²)dx Let x = a sinθ: = 4(b/a)∫₀^(π/2) a cosθ · a cosθ dθ = 4ab∫₀^(π/2) cos²θ dθ = 4ab · π/4 = πab ✓

7. Hyperbola

Standard Hyperbola: x²/a² − y²/b² = 1
Foci(±c, 0) where c² = a² + b²
Vertices(±a, 0)
Eccentricitye = c/a > 1
Asymptotesy = ±(b/a)x
Directricesx = ±a/e
Latus rectum2b²/a
Conjugate hyperbola−x²/a² + y²/b² = 1 (foci on y-axis)
Focal radii|SP − S'P| = 2a (definition)
Rectangular hyperbolaxy = c² (asymptotes perpendicular)

Rectangular Hyperbola xy = c²

Parametric: (ct, c/t)
Tangent at (ct, c/t):
x/t² + yt = 2c   or   x + yt² = 2ct
Normal at (ct, c/t):
xt³ − yt − ct⁴ + c = 0
Asymptotes: x=0, y=0
e = √2 always
Example 1 — Hyperbola foci
For x²/16 − y²/9 = 1, find foci, eccentricity, and asymptotes.
a²=16, b²=9 → a=4, b=3, c=√(16+9)=5 Foci: (±5, 0). Eccentricity: e = 5/4. Asymptotes: y = ±(3/4)x → 3x−4y=0 and 3x+4y=0.
Example 2 — Conjugate hyperbola
If sum of reciprocals of eccentricities of a hyperbola and its conjugate is 1, find eccentricities.
For hyperbola: e₁ = √(1+b²/a²); conjugate: e₂ = √(1+a²/b²) 1/e₁ + 1/e₂ = 1. Using identity: 1/e₁² + 1/e₂² = 1 is a known result. Let 1/e₁ = x, 1/e₂ = 1−x. Also 1/e₁² + 1/e₂² = 1: x²+(1−x)² = 1 → 2x²−2x=0 → x(x−1)=0 But both must be between 0 and 1. This system requires more constraints. For e₁=e₂=√2: 1/√2+1/√2=√2≠1. So the relationship depends on a, b.
Example 3 — Rectangular hyperbola chord
Show that the chord joining (ct₁, c/t₁) and (ct₂, c/t₂) on xy=c² has equation: x + yt₁t₂ = c(t₁+t₂).
Slope of chord = (c/t₁ − c/t₂)/(ct₁ − ct₂) = c(t₂−t₁)/(t₁t₂) / c(t₁−t₂) = −1/(t₁t₂) Equation: y − c/t₁ = (−1/(t₁t₂))(x − ct₁) y·t₁t₂ − ct₂ = −x + ct₁ → x + yt₁t₂ = c(t₁+t₂) ✓

8. Practice Problems (25 Questions)

Lines & Basics

  1. Find the equation of the line through (3,−2) perpendicular to 4x−3y+5=0.
  2. Show that (1,2), (3,0), (−1,4) are collinear.
  3. Find the image of (3,4) in the line x+2y−5=0.
  4. Two lines through origin make equal angles with x+y=1. Write their equations.
  5. Find k if 3x+ky−2=0 and 2x+y+3=0 are parallel.
  6. Find the area of the triangle bounded by x=0, y=0 and 3x+4y=12.
  7. Find the angle between lines 2x−y+3=0 and x+3y−2=0.
  8. Find the locus of point equidistant from lines x+y=1 and x+y=3.

Circles

  1. Find the circle with diameter endpoints (−3,2) and (1,4).
  2. Find the length of the chord of x²+y²=25 cut off by 3x−4y=0.
  3. Show that x²+y²−6x+4y=0 and x²+y²+2x−8y+8=0 intersect at two points. Find the common chord.
  4. Find the two common tangents to circles x²+y²=4 and x²+(y−3)²=1.

Parabola

  1. Find the focus, directrix, and latus rectum of y² = 12x.
  2. Find the equation of the normal to y² = 4x at (1,2).
  3. Find the chord of contact from (−1,1) to the parabola y² = 8x.
  4. If t₁ and t₂ are the parameters of two normals to y²=4ax that intersect on the parabola, show t₁t₂ = 2.
  5. Find the locus of midpoints of chords of y²=4ax that pass through the focus.

Ellipse & Hyperbola

  1. Find the equation of the ellipse with foci (±2,0) and e = 2/3.
  2. Prove that the sum of squares of the distances of any point on the ellipse from the two foci is 2a²+2b²−2a²e² = 2b²+c² (simplify).
  3. Find the tangent to x²/4+y²/3=1 parallel to y=2x+1.
  4. Find the condition for lx+my+n=0 to be tangent to x²/a²−y²/b²=1.
  5. For xy=16, find the tangent at (4,4). Also find where it meets the coordinate axes.
  6. Find the asymptotes of 2x²+5xy+3y²+6x+7y+4=0 (pair of lines).
  7. An ellipse has semi-major axis 5, semi-minor axis 3. Find the equation if centre is at origin, major axis along x-axis. Then find eccentricity.
  8. Find the eccentricity of a hyperbola if the angle between asymptotes is 60°.
