1. Algebra

1.1 Quadratic Equations

Standard form: ax² + bx + c = 0 (a ≠ 0).

Quadratic Formula

x = (-b ± √(b² - 4ac)) / 2a

Discriminant D = b² - 4ac

DiscriminantNature of RootsExample
D > 0Two distinct real rootsx² - 5x + 6 = 0, D = 1
D = 0Two equal real roots (repeated)x² - 4x + 4 = 0, D = 0
D < 0Two complex conjugate rootsx² + x + 1 = 0, D = -3

Vieta's Formulas (roots α, β)

  • Sum of roots: α + β = -b/a
  • Product of roots: αβ = c/a
  • Sum of squares: α² + β² = (α+β)² - 2αβ
  • Difference squared: (α-β)² = (α+β)² - 4αβ

1.2 Sequences & Series

Arithmetic Progression (AP)

  • General term a_n: a + (n-1)d
  • Sum of n terms S_n: n/2 × [2a + (n-1)d] = n/2 × (a + l)
  • Arithmetic Mean AM: (a + b) / 2
  • n-th term from end: l - (n-1)d

Geometric Progression (GP)

  • General term a_n: a × r^(n-1)
  • Sum (finite, r≠1): S_n = a(r^n - 1) / (r - 1)
  • Sum (infinite, |r|<1): S = a / (1 - r)
  • Geometric Mean GM: √(ab)

Harmonic Progression (HP)

  • HM of a and b: 2ab / (a + b)
  • Key inequality: AM ≥ GM ≥ HM (for positive numbers)

Sum of Special Series

Σn = n(n+1)/2
Σn² = n(n+1)(2n+1)/6
Σn³ = [n(n+1)/2]²

1.3 Binomial Theorem

(a + b)^n = Σ C(n,r) · a^(n-r) · b^r, r from 0 to n
  • General term T_(r+1): C(n,r) · a^(n-r) · b^r
  • Sum of all coefficients: Set a=b=1 to get 2^n
  • Sum of even-position: C(n,0)+C(n,2)+... = 2^(n-1)
  • Pascal's rule: C(n,r-1) + C(n,r) = C(n+1,r)

1.4 Permutations & Combinations

  • Permutation nPr: n! / (n-r)!
  • Combination nCr: n! / [r! × (n-r)!]
  • Symmetry: nCr = nC(n-r)
  • Sum property: nC0 + nC1 + ... + nCn = 2^n
  • Circular permutation: (n-1)!
  • Necklace permutation: (n-1)!/2
  • Multinomial coefficient: n! / (n1! n2! ... nk!)
Note: nC(r-1) + nCr = (n+1)Cr (Pascal's identity)

1.5 Complex Numbers

A complex number z = a + bi where i = √(-1), i² = -1.

  • Modulus |z|: √(a² + b²)
  • Conjugate z̅: a - bi
  • Argument arg(z): tan^(-1)(b/a)
  • Polar form: r(cosθ + i sinθ) = re^(iθ)
  • Product moduli: |z1×z2| = |z1|×|z2|
  • Product argument: arg(z1z2) = arg(z1) + arg(z2)
  • z × z̅: |z|² = a² + b²
De Moivre's Theorem: (cosθ + i sinθ)^n = cos(nθ) + i sin(nθ)
Euler's Identity: e^(iπ) + 1 = 0 (most beautiful equation in mathematics)
nth roots of unity: ωk = e^(2πik/n), k = 0,1,...,n-1
Sum of nth roots of unity = 0

1.6 Inequalities

  • AM-GM (2 vars): (a+b)/2 ≥ √(ab) for a,b > 0; equality iff a = b
  • AM-GM (n vars): (a1+a2+...+an)/n ≥ (a1×a2×...×an)^(1/n)
  • Cauchy-Schwarz: (a1²+a2²)(b1²+b2²) ≥ (a1b1+a2b2)²
  • Triangle inequality: |a+b| ≤ |a| + |b|
  • Power Mean chain: QM ≥ AM ≥ GM ≥ HM (all positive)

2.2 Trigonometric Identities

Pythagorean Identities

sin²θ + cos²θ = 1
1 + tan²θ = sec²θ
1 + cot²θ = csc²θ

Sum & Difference Formulas

  • sin(A ± B): sinA cosB ± cosA sinB
  • cos(A ± B): cosA cosB ∓ sinA sinB
  • tan(A + B): (tanA + tanB) / (1 - tanA tanB)
  • tan(A - B): (tanA - tanB) / (1 + tanA tanB)

Double Angle Formulas

  • sin 2A: 2 sinA cosA
  • cos 2A (form 1): cos²A - sin²A
  • cos 2A (form 2): 1 - 2sin²A
  • cos 2A (form 3): 2cos²A - 1
  • tan 2A: 2tanA / (1 - tan²A)

Half Angle Formulas

  • sin²(A/2): (1 - cosA) / 2
  • cos²(A/2): (1 + cosA) / 2
  • tan(A/2): sinA / (1 + cosA) = (1 - cosA) / sinA

Product-to-Sum Formulas

  • 2 sinA cosB: sin(A+B) + sin(A-B)
  • 2 cosA cosB: cos(A-B) + cos(A+B)
  • 2 sinA sinB: cos(A-B) - cos(A+B)

Sum-to-Product Formulas

  • sinC + sinD: 2 sin((C+D)/2) cos((C-D)/2)
  • sinC - sinD: 2 cos((C+D)/2) sin((C-D)/2)
  • cosC + cosD: 2 cos((C+D)/2) cos((C-D)/2)
  • cosC - cosD: -2 sin((C+D)/2) sin((C-D)/2)

t-Formulas (t = tan(A/2))

sinA = 2t/(1+t²)     cosA = (1-t²)/(1+t²)     tanA = 2t/(1-t²)

2.3 Triangle Formulas

  • Sine Rule: a/sinA = b/sinB = c/sinC = 2R
  • Cosine Rule: a² = b² + c² - 2bc cosA
  • Area (trig): Δ = (1/2)ab sinC
  • Heron's Formula: Δ = √[s(s-a)(s-b)(s-c)], s = (a+b+c)/2
  • Inradius r: r = Δ/s
  • Circumradius R: R = abc / (4Δ)

3. Calculus

3.1 Limits

Standard Limits

  • lim(x→0) (sin x)/x: 1
  • lim(x→0) (tan x)/x: 1
  • lim(x→0) (e^x - 1)/x: 1
  • lim(x→0) (a^x - 1)/x: ln a
  • lim(x→0) (1+x)^(1/x): e
  • lim(x→∞) (1+1/x)^x: e
  • lim(x→0) ln(1+x)/x: 1
  • lim(x→a) (x^n - a^n)/(x-a): na^(n-1)
L'Hôpital's Rule: If f(x)/g(x) → 0/0 or ∞/∞ as x→a,
then lim f(x)/g(x) = lim f'(x)/g'(x) (if the latter limit exists)

Sandwich (Squeeze) Theorem: If f(x) ≤ g(x) ≤ h(x) near a, and lim f(x) = lim h(x) = L, then lim g(x) = L.

3.2 Differentiation

Basic Rules

  • Power Rule: d/dx (x^n) = n x^(n-1)
  • Product Rule: (uv)' = u'v + uv'
  • Quotient Rule: (u/v)' = (u'v - uv') / v²
  • Chain Rule: d/dx [f(g(x))] = f'(g(x)) × g'(x)

Applications of Derivatives

  • Tangent slope at x0: m = f'(x0)
  • Normal slope: -1/f'(x0)
  • Increasing function: f'(x) > 0 on interval
  • Decreasing function: f'(x) < 0 on interval
  • Critical points: f'(c) = 0 or undefined
  • Local max (2nd deriv): f'(c) = 0 and f''(c) < 0
  • Local min (2nd deriv): f'(c) = 0 and f''(c) > 0
  • Concave up: f''(x) > 0
  • Inflection point: f''(x) = 0 and sign changes

Methods of Integration

  • Substitution: Let u = g(x); ∫f(g(x))g'(x)dx = ∫f(u)du
  • By Parts (LIATE): ∫u dv = uv - ∫v du
  • LIATE order: Log, Inverse trig, Algebraic, Trig, Exponential
  • Partial fractions: Decompose rational function before integrating

Definite Integral Properties

  • Reversal: ∫[a to b] f = -∫[b to a] f
  • Additive interval: ∫[a to c] = ∫[a to b] + ∫[b to c]
  • Even function: ∫[-a to a] f = 2∫[0 to a] f
  • Odd function: ∫[-a to a] f = 0
  • King's property: ∫[a to b] f(x)dx = ∫[a to b] f(a+b-x)dx
  • Period: ∫[0 to nT] f = n ∫[0 to T] f for periodic f with period T

Applications of Integration

  • Area under curve: A = ∫[a to b] |f(x)| dx
  • Area between curves: A = ∫[a to b] [f(x) - g(x)] dx
  • Volume (disk method): V = π ∫[a to b] [f(x)]² dx
  • Volume (shell method): V = 2π ∫[a to b] x f(x) dx

3.4 Differential Equations

  • Order: Highest derivative present
  • Degree: Power of highest-order derivative (polynomial form)
  • Variable separable: f(y)dy = g(x)dx; integrate both sides
  • Homogeneous (y=vx): If dy/dx = F(y/x), substitute y = vx
  • Linear 1st-order form: dy/dx + P(x)y = Q(x)
  • Integrating Factor IF: e^(∫P dx)
  • General solution: y × IF = ∫ Q × IF dx + C

4. Coordinate Geometry

4.1 Straight Lines

  • Distance formula: d = √[(x2-x1)² + (y2-y1)²]
  • Midpoint: M = ((x1+x2)/2, (y1+y2)/2)
  • Section formula (m:n internal): ((mx2+nx1)/(m+n), (my2+ny1)/(m+n))
  • Slope: m = (y2-y1)/(x2-x1) = tan θ
  • Slope-intercept form: y = mx + c
  • Point-slope form: y - y1 = m(x - x1)
  • Two-point form: (y-y1)/(y2-y1) = (x-x1)/(x2-x1)
  • Intercept form: x/a + y/b = 1
  • Normal form: x cosα + y sinα = p
  • Angle between two lines: tanθ = |m1-m2| / (1+m1 m2)
  • Distance: point to line: d = |ax0+by0+c| / √(a²+b²)
  • Parallel condition: m1 = m2
  • Perpendicular condition: m1 × m2 = -1

4.2 Circles

  • Standard form: (x-h)² + (y-k)² = r²
  • General form: x²+y²+2gx+2fy+c = 0
  • Center: (-g, -f)
  • Radius: √(g² + f² - c)
  • Tangent at (x1,y1): xx1 + yy1 + g(x+x1) + f(y+y1) + c = 0
  • Tangent length from P: L = √(x1²+y1²+2gx1+2fy1+c)
  • Power of point P: L² = x1²+y1²-r² (for circle centered at origin)

  • Eccentricity e: e = c/a, 0 < e < 1

  • Relation: b² = a²(1 - e²)

  • Foci: (±c, 0) where c² = a² + b²
  • Eccentricity e: e = c/a, e > 1
  • Asymptotes: y = ±(b/a)x
  • Relation: b² = a²(e² - 1)
  • Diff. of focal radii: |r1 - r2| = 2a (constant)

5. 3D Geometry & Vectors

5.1 Direction Cosines & Ratios

  • DCs (l, m, n): cosα, cosβ, cosγ (angles with x, y, z axes)
  • Identity: l² + m² + n² = 1
  • DRs (a, b, c) to DCs: l = a/√(a²+b²+c²), similarly m, n
  • Distance (3D): d = √[(x2-x1)²+(y2-y1)²+(z2-z1)²]

5.2 Vector Operations

  • Magnitude |a|: √(a1² + a2² + a3²)
  • Unit vector: â = a / |a|
  • Dot product a·b: |a||b|cosθ = a1b1 + a2b2 + a3b3
  • Cross product |a×b|: |a||b|sinθ (direction: right-hand rule)
  • Cross product components: (a2b3-a3b2, a3b1-a1b3, a1b2-a2b1)
  • Area of parallelogram: |a×b|
  • Area of triangle: (1/2)|a×b|
  • Scalar triple product [a,b,c]: a·(b×c)
  • Volume of parallelepiped: |[a,b,c]|
  • Coplanar test: [a,b,c] = 0

5.3 Lines & Planes in 3D

  • Line (vector form): r = a + λb
  • Line (symmetric): (x-x1)/l = (y-y1)/m = (z-z1)/n
  • Plane (vector): r·n = d
  • Plane (Cartesian): ax + by + cz = d
  • Plane (intercept form): x/a + y/b + z/c = 1
  • Dist. point to plane: |ax0+by0+cz0-d| / √(a²+b²+c²)
  • Angle between line & plane: sinθ = |l·a+m·b+n·c| / (1 × √(a²+b²+c²))
  • Dist. between skew lines: |(a2-a1)·(d1×d2)| / |d1×d2|

6.2 Determinants

2×2: det(A) = ad - bc, for A = [[a,b],[c,d]]

3×3 (cofactor expansion along row 1):
det(A) = a11(a22 a33 - a23 a32) - a12(a21 a33 - a23 a31) + a13(a21 a32 - a22 a31)

Properties of Determinants

  • Row/column swap: det changes sign
  • Multiply row by k: det multiplied by k
  • Two identical rows: det = 0
  • Row operation R_i + k R_j: det unchanged
  • det(AB): det(A) × det(B)
  • det(A^T): det(A)
  • det(kA) for n×n: k^n det(A)
  • Singular: det(A) = 0 (no inverse exists)

6.3 Inverse & Linear Systems

  • Matrix inverse: A^(-1) = adj(A) / det(A)
  • Adjoint adj(A): Transpose of cofactor matrix of A
  • Cramer's Rule x_i: det(A_i) / det(A), where A_i has b in column i
  • Unique solution: det(A) ≠ 0
  • Infinite solutions: det(A) = 0, rank(A) = rank([A|b])
  • No solution: rank(A) ≠ rank([A|b])

7. Statistics & Probability

7.1 Measures of Central Tendency

  • Mean (ungrouped): x̄ = Σxi / n
  • Weighted mean: x̄ = Σwi xi / Σwi
  • Combined mean: x̄12 = (n1 x̄1 + n2 x̄2) / (n1 + n2)
  • Median (odd n): Middle value of sorted data
  • Median (even n): Average of two middle values
  • Mode: Most frequently occurring value

7.2 Measures of Dispersion

  • Range: Maximum - Minimum
  • Mean Deviation (from mean): MD = Σ|xi - x̄| / n
  • Variance: σ² = Σ(xi - x̄)² / n
  • Variance (shortcut): σ² = Σxi²/n - (x̄)²
  • Standard deviation: σ = √(σ²)
  • Coeff. of variation: CV = (σ / x̄) × 100 %

7.3 Probability Theorems

  • Classical probability: P(E) = n(E) / n(S)
  • Complement rule: P(A') = 1 - P(A)
  • Addition rule: P(A∪B) = P(A) + P(B) - P(A∩B)
  • Mutually exclusive: P(A∪B) = P(A) + P(B)
  • Conditional probability: P(A|B) = P(A∩B) / P(B)
  • Independent events: P(A∩B) = P(A) × P(B)
  • Total probability: P(B) = Σ P(B|Ai) P(Ai)
  • Bayes' theorem: P(Ai|B) = P(B|Ai)P(Ai) / ΣP(B|Aj)P(Aj)

7.4 Probability Distributions

Binomial Distribution B(n, p)

P(X = r) = C(n,r) p^r q^(n-r), where q = 1 - p, r = 0, 1, ..., n
Mean μ = np
Variance σ² = npq
Standard deviation σ = √(npq)

Normal Distribution N(μ, σ²)

  • Standard normal Z: Z = (X - μ) / σ
  • 68-95-99.7 rule: P(μ±σ)≈68%, P(μ±2σ)≈95%, P(μ±3σ)≈99.7%
  • Shape: Bell-shaped, symmetric around μ
  • Mean = Median = Mode: All equal μ
  • Total area: = 1

Common Mistakes and How to Avoid Them

  • Forgetting ± when taking square roots: always write both cases
  • Confusing sin(A+B) with sinA + sinB: these are NOT equal
  • Skipping chain rule for composite functions in differentiation
  • Not adding constant C in indefinite integration
  • Forgetting to check domain/range in inverse trigonometry problems
  • Sign errors in determinant cofactor expansion (use checkerboard +/-/+ pattern)
  • Wrong limits or substitution in definite integrals

Problem 1 — Algebra: Quadratic Roots
If α and β are the roots of 2x² - 5x + 3 = 0, find α² + β².
Given: 2x² - 5x + 3 = 0 (a=2, b=-5, c=3) By Vieta's formulas: α + β = -b/a = 5/2 αβ = c/a = 3/2 Use the identity: α² + β² = (α+β)² - 2αβ = (5/2)² - 2 × (3/2) = 25/4 - 3 = 25/4 - 12/4 = 13/4
Answer: α² + β² = 13/4
Problem 2 — Trigonometry: Identity Proof
Prove that: sin(A+B) × sin(A-B) = sin²A - sin²B
LHS = sin(A+B) × sin(A-B) = (sinA cosB + cosA sinB)(sinA cosB - cosA sinB) = sin²A cos²B - cos²A sin²B = sin²A(1 - sin²B) - (1 - sin²A)sin²B = sin²A - sin²A sin²B - sin²B + sin²A sin²B = sin²A - sin²B = RHS ✓
Proved: LHS = RHS
Problem 3 — Calculus: Product Rule Differentiation
Find dy/dx if y = x² sin(3x).
Let u = x² and v = sin(3x) u' = 2x v' = 3cos(3x) [chain rule: d/dx sin(3x) = cos(3x) × 3] By product rule: dy/dx = u'v + uv' = (2x)(sin 3x) + (x²)(3 cos 3x) = 2x sin(3x) + 3x² cos(3x)
dy/dx = 2x sin(3x) + 3x² cos(3x)
Problem 4 — Integration: By Parts (LIATE)
Evaluate ∫ x e^x dx.
By LIATE rule, choose: u = x (Algebraic) → du = dx dv = eˣ dx → v = eˣ Apply integration by parts: ∫u dv = uv - ∫v du = x·eˣ - ∫eˣ dx = xeˣ - eˣ + C = eˣ(x - 1) + C
∫ x e^x dx = e^x (x - 1) + C
Problem 5 — Coordinate Geometry: Circle Through 3 Points
Find the equation of the circle passing through (1, 2), (3, 4), and (5, 2).
Let circle equation: x² + y² + 2gx + 2fy + c = 0 Substituting (1,2): 1+4+2g+4f+c = 0 → 2g+4f+c = -5 ...(1) Substituting (3,4): 9+16+6g+8f+c = 0 → 6g+8f+c = -25 ...(2) Substituting (5,2): 25+4+10g+4f+c = 0 → 10g+4f+c = -29 ...(3) (2) - (1): 4g + 4f = -20 → g + f = -5 ...(4) (3) - (1): 8g = -24 → g = -3 From (4): f = -5 - (-3) = -2 From (1): c = -5 - 2(-3) - 4(-2) = -5+6+8 = 9 Circle equation: x²+y²-6x-4y+9 = 0 Center: (3, 2), Radius = √(9+4-9) = √4 = 2

Frequently Asked Questions

What are the most important math formulas for JEE?
Key JEE math formulas include calculus (integration standard forms, differentiation rules, definite integral properties), trigonometry identities (sum/difference, double angle, product-to-sum), coordinate geometry (tangent to circle/ellipse, distance from point to line), and algebra (quadratic formula, Vieta's formulas, binomial theorem). Calculus and Coordinate Geometry together account for nearly 50% of marks.
How to prepare mathematics for Gaokao?
Focus on mastering functions and derivatives, sequences (AP/GP), analytic geometry (conics), and probability. Practice Gaokao past papers from 2015–2024 — problems follow very predictable formats. The exam rewards accuracy and complete step-by-step solutions; partial credit is given even for incorrect final answers if your working is correct.
What is the quadratic formula and when should I use it?
The quadratic formula x = (-b ± √(b²-4ac)) / 2a solves any equation ax² + bx + c = 0. Use it when: (1) the equation cannot be factored easily, or (2) you need exact values of roots. Always compute the discriminant D = b²-4ac first to determine the nature of roots before applying the formula.
What is the AM-GM inequality and how is it used in JEE?
The AM-GM inequality states (a+b)/2 ≥ √(ab) for positive a, b. It generalizes to n variables. In JEE, it is used to find minimum or maximum values of algebraic expressions without calculus. For example, to find the minimum of a + 9/a for a > 0: AM-GM gives a + 9/a ≥ 2√(9) = 6, with equality when a = 3.
How to remember integration by parts and the LIATE rule?
Integration by parts: ∫u dv = uv - ∫v du. Choose u using the LIATE priority: Logarithm > Inverse trig > Algebraic > Trigonometric > Exponential. The function that appears first in this list is chosen as u. Example: ∫x² ln x dx — ln x is Logarithm (highest priority), so u = ln x, dv = x² dx. This minimizes the complexity of ∫v du.
